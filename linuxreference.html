Here are commands.

Command Command Syntax Description Example
man man command Displays the help page for the specified command. man echo
echo echo string Displays string in terminal. echo hello world
cd cd directory Changes the directory to the specified directory. cd Documents
ls ls [directory -al] Lists all files and directories inside a directory. -a lists all files. -l lists details about the file. ls Pictures -al
history history Displays command history. history
touch touch file-name Creates a file with the specified file name in the current directory. touch test
pwd pwd Prints working or current directory. pwd
rm rm file-or-directory [-rf] Removes the specified file or directory. -r removes recursively (everything inside a directory). -f forces removal. rm -rf Documents
mv mv file-or-directory destination-file-or-directory Moves a file or directory to a different location. Can be used to rename files. mv test Documents
cp cp file-or-directory destination-file-or-directory Copies one file or directory to another. cp test Desktop
scp scp file_name destinationuser@remotehost:path Securely copies file from one device to another. scp test sp1a@rpi01:~/Documents
